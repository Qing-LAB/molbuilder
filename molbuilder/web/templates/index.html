molbuilder

Build a 3-D molecule from a peptide / DNA / RNA sequence, a SMILES string, or a chemical name — and turn it into a SIESTA .fdf or PySCF input.

Build Watch

1. Build / Load

… or load an existing .xyz / .pdb file

Skip the build step and start from a structure you already have (e.g. one written out by a previous SIESTA / VMD / PyMOL session). The loaded geometry feeds the viewer and the input generator below.

2. Generate input (optional)

Defaults reproduce a standard small-system DFT setup (DZP basis, 300 Ry mesh, GGA-PBE, SCF-only or CG relaxation). Tweak whatever you need; everything not shown stays at the SIESTA default.
